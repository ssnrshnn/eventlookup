# Event ID 4778 - RDP Session Reconnected

## Overview
**Event Name:** RDP Session Reconnected  
**Category:** Authentication  
**Criticality:** Medium  
**Log Source:** Security  

## Description
A session was reconnected to a Window Station. Monitor for RDP session activity.
