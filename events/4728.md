# Event ID 4728 - Member Added to Global Group

## Overview
**Event Name:** Member Added to Global Group  
**Category:** Account Management  
**Criticality:** High  
**Log Source:** Security  

## Description

## Investigation Tips

- Verify authorization for group membership changes

- Review timing of membership changes
