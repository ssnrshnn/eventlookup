# Event ID 4616 - System Time Changed

## Overview
**Event Name:** System Time Changed  
**Category:** System  
**Criticality:** Medium  
**Log Source:** Security  

## Description
The system time was changed. May indicate timestamp manipulation attempts.

- Review time change amount and direction
